display: flex

A propriedade display: flex deve ser a primeira a ser declarada no CSS. A parti dela é que o CSS vai entender que precisa estilizar o conteúdo do contêiner com a Caixa Flexível (Flexbox)

Exemplo:

Valores padrão das propriedades de Flexbox
propriedade valor
flex-direction row
justify-content flex-start
align-content flex-start

justify-content

A propriedade justify-content pode ser usada para justificar o conteúdo do elemento contêiner em seu eixo primário, ou seja, eixo x (row, row-reverse) ou eixo y (colum, colum-reverse).

center

O valor "center" vai alinhar o conteúdo ao centro do eixo primário

flex-direction

A propriedade CSS flex-direction define como os itens flexíveis são colocados no contêiner flexível, definindo o eixo principal e a direção (normal ou invertido).

row

Valor padrão. Os itens flexíveis são exibidos horizontalmente, como uma linha
